# Libo Chen Lab Homepage

This is a redesigned light academic homepage for GitHub Pages.

## Files
- `index.html` — replace the existing homepage file with this file.
- `CNAME` — contains `www.libochenlab.com`. Keep this file in your GitHub repository root.

## Upload location
Upload `index.html` to the root of:

`Libo-Chen.github.io/`

The repository should look like:

```
Libo-Chen.github.io/
├── index.html
└── CNAME
```

## Before publishing
Edit the following in `index.html`:
- Replace `your.email@uu.se` with your actual contact email.
- Replace placeholder publication links.
- Add real news items.
- Add group members and photos when available.
